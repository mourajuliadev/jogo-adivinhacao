# Meu Primeiro Projeto em Java 💻

Este projeto marca o início da minha jornada no desenvolvimento com Java.

## Sobre o projeto

Este é um jogo de adivinhação desenvolvido como desafio prático durante meus estudos na plataforma Alura.

O programa funciona da seguinte forma:

* Gera um número aleatório entre 0 e 100
* O usuário tem até 5 tentativas para acertar
* A cada tentativa, o sistema informa se o número é maior ou menor

## Objetivo

Praticar conceitos fundamentais de Java, como:

* Entrada de dados com Scanner
* Geração de números aleatórios com Random
* Estruturas de repetição (while)
* Condicionais (if/else)
* Lógica de programação

## Tecnologias utilizadas

* Java
* IntelliJ IDEA

## Como executar

1. Abra o projeto no IntelliJ
2. Execute a classe principal
3. Interaja pelo console
